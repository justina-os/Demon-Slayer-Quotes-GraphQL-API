# Demon Slayer Quotes GraphQL API

A FastAPI + Strawberry GraphQL API for fetching Demon Slayer quotes.

## Features

- Random quotes
- Quotes by character
- Quotes by category
- MySQL database integration
- GraphQL endpoint

## Tech Stack

- FastAPI
- Strawberry GraphQL
- MySQL

## Run Locally

```bash
pip install -r requirements.txt
uvicorn main:app --reload
